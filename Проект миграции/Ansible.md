Установка 
=========

Установка необходима только на машине с которой ведёться управление серверами

Mac:

    brew install ansible

Ubuntu:
  
    sudo add-apt-repository -y ppa:rquillo/ansible
    sudo apt-get update
    sudo apt-get install ansible -y
